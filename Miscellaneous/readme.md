# Green Earth Initiative Website

## Project Overview

The Green Earth Initiative website is a multi-page, responsive website for an NGO focused on environmental sustainability. The website showcases the organization's missions, gallery, contact information, and provides users with ways to engage with their projects.

## Features

* Multi-page website (Home, About, Missions, Gallery, Contact, References)
* Responsive design for mobile and desktop screens
* Interactive gallery with lightbox functionality
* Embedded Google Form for contact and volunteer sign-ups
* Interactive map showing project locations
* SEO optimized with meta tags, keywords, and descriptions
* Scroll-to-top button for improved user experience

## Technologies Used

* HTML5
* CSS3 (external stylesheet)
* JavaScript (lightbox and interactivity)
* Google Forms for contact/volunteer submissions
* Google Maps embed

## Installation

1. Clone or download the repository.
2. Open any of the `.html` files in a web browser.
3. Ensure `css/style.css` and `js/script.js` are in the correct folders.

## Usage

* Navigate between pages using the top menu.
* View the gallery and click images to enlarge using the lightbox.
* Contact the organization or sign up to volunteer via the embedded Google Form.
* Check mission details and interactive map on the Missions page.
